# Library Database Project

## Course
Database Systems

## Description
This project implements a library database management system.
The database models books, authors, volumes, customers and purchases.

The system supports:
- Books with multiple volumes
- Customers purchasing books
- Relationships between books and authors
- SQL queries with JOIN, GROUP BY and subqueries

## Technologies
- MySQL
- SQL
